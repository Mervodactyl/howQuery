AJAX not so scary after all...

One of the best things about jQuery is how it simplifies woking with AJAX, as it provides high level wrapper functions for retrieveing data, working with different types, handling errors, and monitoring the AJAX process

  • ajax() : is used to perform generic AJAX requests
  • load() : loads the AJAX content directly into a page element
